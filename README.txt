Modern Sonic Framework for GameMaker 8.2

HOW TO PLAY:
Basic Controls:
Move left/right = Arrow keys/Analog stick
Jump = A
Airdash = A + A(When there is no target)
Homing Attack  = A + A(When there is a target)
Boost = S/B
Lightspeed dash = F/Y
Stomp = Down arrow key/Analog stick
Spindash = Down + Jump
Spring Tricks = Direction + Boost(When hitting a spring)
Grind duck = Down arrow key/Analog stick(When on a rail)
